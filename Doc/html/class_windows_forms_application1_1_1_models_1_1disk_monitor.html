Task Manager  1.0
Pantalla de monitoreo similar a la pestaña RENDIMIENTO del administrador de tareas de Windows en donde se visualice el uso de CPU, MEMORIA, DISCO y RED. Cada uno de los indicadores debe funcionar y refrescar la información por medio de un thread distinto.
